This course introduces the students to relational databases. The important design concepts and the common standards to which all database packages adhere are discussed. Students also learn to query for information and generate reports using a popular database package. (Prerequisites: CA110 Computer Concepts and TY101 Keyboarding I)

Five Things I Learned in this Course

  • What SQL is
  • How to create tables and columns
  • How to select and update columns and data in columns
  • How to join multiple tables
  • How to create views
The assignment was to list the product line the product name and how many have been ordered of each product.
I selected the columns labeled productLine and productName from the the table called products, and renamed them to make the column names look nice. Then i grab the column labeled quantityOrdered from table orderdetails. To do that, I needed to make a join between the tables named orderdetails and products. They both shared the column productCode. Finally I sumed the quantityOrdered column, and grouped the table by productName.
